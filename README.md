# UM Survey Auto-Fill (ETS)

Chrome extension that automates UM performance evaluation surveys. Select a rating, optionally add a comment, and fill all questions in one click.

## Install

1. Download/clone this repo
2. Go to `chrome://extensions/`
3. Enable **Developer mode** (top-right)
4. Click **Load unpacked** and select this folder

## Usage

The side panel opens automatically when you navigate to a survey page. You can also click the extension icon to open it manually.

1. Navigate to a UM evaluation survey page (`mis.umin.edu.ph/oPEAS/...`)
2. Select a rating (5 = Excellent, 4 = Very Satisfactory, 3 = Satisfactory, 2 = Below Satisfactory, 1 = Needs Improvement)
3. (Optional) Toggle **Include Comment** and type a message
4. (Optional) Toggle **Auto-Submit** to submit automatically after filling
5. Click **Fill Survey**

Your preferences are saved between sessions.

## Distribute

Zip this folder and share it. Recipients install it the same way via **Load unpacked**.
